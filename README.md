# Random Cat API

Projeto interativo desenvolvido com JavaScript para praticar consumo de APIs.

## 🚀 Funcionalidades
- Requisição assíncrona via fetch
- Atualização dinâmica do DOM
- Exibição de imagem aleatória a cada clique

## 🛠 Tecnologias
- HTML5
- CSS3
- JavaScript
- API externa
